ColliMap

Carica mappa immagine e JSON opzionale. Se il JSON non esiste, puoi generarlo da zero e scaricarlo già pronto.

1. Input file

Se carichi solo i layer immagine, la UI crea una base JSON usando tileSize, righe e colonne impostate qui sotto.

1B. Config gioco (diretto)

Questi pulsanti modificano direttamente la config mappa del gioco via API (/api/collimap/*). In assenza API resta disponibile il flusso file manuale.

2. Metadati mappa

Se l'immagine misura 1536×1024 con tileSize 64, il layout corretto è 24×16.

3. Collisioni

La collisionMap descrive il layer 2 di gameplay: solo qui il PG viene bloccato. Trascina sul canvas per dipingere. Tasto destro = applica il pennello opposto. Scorciatoie: B bloccato, L libero, T toggle, P spawn, Ctrl/Cmd+Z undo, Ctrl/Cmd+Y redo.

4. Spawn e simulazione

Usa Seleziona spawn e clicca una tile libera, oppure inserisci X/Y manualmente. Durante la simulazione muovi il PG con WASD. Il movimento rispetta le collisioni e parte dallo spawn salvato nel JSON.

5. JSON avanzato

Questo campo viene preservato nel file finale. Puoi usarlo per interactionPoints, transitionPoints, healTile e altri dati del nodo points.

6. Righe collisionMap

Ogni riga deve avere esattamente lo stesso numero di colonne del layout.
